## 间距基准值

### 元素间距（动画 delay 基准）
| 场景 | 最小 delay | 推荐 delay | 最大 |
|------|-----------|-----------|------|
| 标题 → 主角 | 300ms | 400ms | 600ms |
| 主角 → 配角 | 200ms | 250ms | 400ms |
| 配角 → 装饰 | 100ms | 150ms | 250ms |
| 同级元素间 | 50ms | 100ms | 150ms |

### 页面内边距
| 断点 | 内边距 |
|------|--------|
| 桌面（>= 1024px） | 80px |
| 平板（768px - 1023px） | 48px |
| 移动（< 768px） | 24px |

## 动画规范

### 入场顺序（必须严格遵守）
```
1. 标题（Title）     — 最先，让用户知道这是什么页面
2. 主角元素（Main）   — 核心内容，用户应第一时间看到
3. 配角元素（Support）— 辅助说明，不抢主角风头
4. 装饰元素（Deco）   — 最后入场，收尾
```

### 动画时长基准
| 元素类型 | 时长范围 | 推荐值 |
|---------|---------|--------|
| 淡入 | 200-400ms | 300ms |
| 位移 | 300-600ms | 400ms |
| 缩放 | 250-500ms | 350ms |
| 组合动画 | 400-800ms | 600ms |

### 入场方向语义
| 元素 | 默认方向 | 备注 |
|------|---------|------|
| 标题 | 从上（translateY: -30px → 0） | 告知页面主题 |
| 副标题 | 从上（translateY: -20px → 0） | 紧随标题 |
| 主角内容 | 从左或右（translateX ±40px → 0） | 视内容方向 |
| 配角文字 | 淡入（opacity: 0 → 1） | 不强调方向 |
| 装饰元素 | 缩放（scale: 0.8 → 1） | 从小到大 |

## 动画陷阱清单

### 高危问题
- ❌ delay = 0ms（所有元素同时入场，无层次感）
- ❌ 主角先于标题入场（用户不知道在看什么）
- ❌ 动画总时长 > 1.5s（节奏拖沓）
- ❌ 所有元素使用同一方向（无视觉引导）

### 中危问题
- ⚠️ 装饰元素 delay 过短（抢主角风头）
- ⚠️ 标题使用 scale 动画（方向语义错误）
- ⚠️ 连续动画无间隔（阅读压力）
- ⚠️ 移动端动画未简化（性能问题）

## 响应式断点

```css
/* 桌面 */
@media (min-width: 1024px) { ... }

/* 平板 */
@media (min-width: 768px) and (max-width: 1023px) { ... }

/* 移动 */
@media (max-width: 767px) { ... }
```

---

## 可访问性标准

- 图片必须有 `alt` 属性
- 颜色对比度：正文 >= 4.5:1，大字 >= 3:1
- 交互元素有 `:focus` 和 `:hover` 状态
- 动画尊重 `prefers-reduced-motion`
