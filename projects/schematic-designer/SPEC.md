# Schematic Designer - 自动原理图设计工具

## 项目概述
一款自动原理图设计与绘制工具，支持 Altium Designer (AD) 和嘉立创 (LCEDA) 格式。

## 技术栈
- **前端**: React 18 + TypeScript + Vite
- **状态管理**: Zustand
- **图形渲染**: SVG + Custom Canvas
- **样式**: TailwindCSS
- **文件解析**: JSZip (用于 AD/LCEDA 格式)

| 地 | GND |
| 连接器 | J (Connector) |
| IC | U (Generic IC) |

### 设计规则

#### 电气规则
```
电压规则:
- 电阻耐压: ≥ 实际电压 × 1.5
- 电容耐压: ≥ 实际电压 × 1.5
- IC 供电电压: 必须符合 datasheet 范围

电流规则:
- 走线电流: ≤ 走线宽度对应电流
- 器件功率: P = V × I, 必须 < 额定功率 × 0.8

连线规则:
- 同网络必须电气连通
- 不同网络禁止意外连接
- 输入引脚不能悬空
```

## 文件格式

### AD 格式 (Altium Designer)
```json
{
  "version": "1.0",
  "type": "schematic",
  "components": [...],
  "nets": [...],
  "wires": [...]
}
```

### LCEDA 格式 (嘉立创)
```json
{
  "docType": "sch",
  "data": {
    "components": [...],
    "paths": [...]
  }
}
```
