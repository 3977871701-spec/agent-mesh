# 经验沉淀模板

此文件由 dg-dev Agent 自动维护，记录开发过程中遇到的问题和解决方案。

## 使用说明
每次完成开发任务或修复 bug 后，将经验追加到此文件。

## 模板
```
## {yymmdd} {hhmm} - {任务/问题名称}

### 遇到的问题
- {问题描述}

### 解决方案
- {解决方案}

### 代码示例
```language
{相关代码片段}
```

### 注意事项
- {需要注意的点}
```

---

## 格式规范
1. 时间戳格式：`yymmdd hhmm`（如 `260508 1430`）
2. 问题描述要具体，避免模糊表述
3. 解决方案要可执行，避免空泛建议
4. 代码示例要精简，保留核心部分

## 示例
```
## 260508 1430 - React useEffect 依赖循环

### 遇到的问题
- 组件陷入 useEffect 依赖循环，不断重新渲染

### 解决方案
- 使用 useCallback 包裹回调函数
- 合理设置依赖数组
- 使用 ref 存储不需要触发渲染的值

### 代码示例
```javascript
const countRef = useRef(0);
useEffect(() => {
  countRef.current += 1;
}, [value]); // 只在 value 变化时执行
```

### 注意事项
- 避免在 useEffect 中直接使用对象作为依赖
- ref 的变化不会触发重新渲染
```
